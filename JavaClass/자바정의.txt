		변수의 사용 이유
1. 간편한 수정
2. 다음 명령에서의 기억
3. 값에 의미 부여

		변수란?
1. 숫자로 시작할 수 없다 
	name="신은서";
	2name="신현서"; 불가능 ---> name2="신현서";
2. 특수문자 사용x $,_만 가능하다
3. 대소문자를 구분한다 
	name,Name 은 다른 변수로 지정가능
4. 이미 사용하고 있는 예약어는 사용불가

		작성 방법의 종류
-1. 헝가리안 표기법
	:변수에 자료형을 구분할 수 있는 접두사 달아표기
	ex) int num; --> int iNum;
-2.카멜 표기법
	:두 개이상의 단어가 사용될 경우, 두번째 단어의 시작은 대문자
	ex) int newName
-3. 파스칼 표기법
	: 클래스명 작성 시, 모든 단어의 첫 글자를 대문자
	ex) class MyClass


	자료형 데이터 타입의 종류
정수형	int	4byte	-2,147,483,648 ~ 2,147,483,647	 
	long	8byte	int 이상
	short	2byte	-32,768 ~ 32,767
실수형	float	4byte	
	double	8byte
문자형	char	2byte
문자열	String	
논리형	boolean	1byte	true or false

	형변환이란?	:변수나 상수의 데이터타입을 다른타입으로 변환하는 것
	형변환의 종류?
1. 자동 형변환(묵시적 형변환)
	:자동으로 타입이 변환 되는 것
2. 명시적 형변환
	:코드 상으로 강제로 형식을 변환시키는 것
	명시적 형변환 방법
	(자료형)변수명;
	ex) char c	= (char)a;

	아스키코드값?
컴퓨터는 문자를 저장할 수 없기 때문에 문자마다 고유 한 숫자값을 부여했는데
그 숫자가 바로 문자의 아스키코드값이다.

	변수의 상수화
1. 상수란?
	:항상 유지되는 수
2. 변수의 상수화
	: 변수에 저장된 값을 변경하지 않고 상수처럼 사용함
3. 상수화 방법
	"final 자료형 변수명=값;"
